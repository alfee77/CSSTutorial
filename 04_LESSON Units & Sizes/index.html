Aboslute Length Units - Pixels (px)

The most frequently used abosolute unit will be the pixel. That said, we'll rarely use absolute lengths, and we definitely shouldnt use it for setting font sizes. If we do, it will over-ride the browser settable sizing (... => Settings => Appearance)

Relative sizing

The use of the rem sizing "dimension" sets the font relative to the root size. So when altering the fault size in the browser (via... => Settings => Appearance => font size), the font size of the page "grows" with it.
